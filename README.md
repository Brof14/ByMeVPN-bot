# 🚀 ByMeVPN Bot - Telegram VPN Sales Bot

> **Стабильный и надёжный Telegram бот для продажи VPN подписок с автоматической выдачей ключей и реферальной системой**

[![Python](https://img.shields.io/badge/python-3.9+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-proprietary-red.svg)](LICENSE)
[![Status](https://img.shields.io/badge/status-production%20ready-green.svg)](https://github.com)

## ✨ Особенности

- 🎁 **Автоматическая выдача ключей** после оплаты
- 💳 **Множественные способы оплаты** (Telegram Stars, YooKassa)
- 🎯 **Реферальная система** (+3 дня за переход, +15 дней за оплату)
- 🛡️ **Надёжная обработка ошибок** с graceful degradation
- 📊 **Детальная статистика** для пользователей и администраторов
- 🔧 **Асинхронная архитектура** для высокой производительности
- 📱 **Поддержка всех устройств** (iOS, Android, Windows, macOS, Linux)

## 🚀 Быстрый старт

### 1. Требования
- Python 3.9+
- SQLite 3
- Доступ к 3x-UI панели
- Telegram Bot Token

### 2. Установка
```bash
# Клонирование
git clone <repository-url>
cd ByMeVPN_bot

# Установка зависимостей
pip install -r requirements.txt

# Конфигурация
cp .env.example .env
nano .env  # Введите ваши данные
```

### 3. Импорт пользователей (ВАЖНО!)
Если у вас есть файл бэкапа пользователей (например `bymevpn_users_20260402_0506.csv`):

```bash
# Импорт пользователей из CSV в базу данных
python import_users.py
```

**⚠️ Важно:** Запустите этот скрипт ОДИН РАЗ перед первым запуском бота, чтобы импортировать всех пользователей!

### 4. Запуск
```bash
# Обычный запуск
python main.py

# Фоновый режим (Linux)
nohup python main.py > bot.log 2>&1 &
```

## 🔧 Технические особенности

### 🚀 Производительность
- **Асинхронная архитектура**: Все операции неблокирующие
- **Connection pooling**: Оптимизация внешних запросов
- **Кэширование**: Частые данные в памяти
- **Batch операции**: Групповая обработка

### 🛡️ Надёжность
- **Graceful degradation**: Работа при частичных сбоях
- **Retry механизмы**: Повторные попытки с backoff
- **Idempotency**: Защита от дубликатов
- **Circuit breaker**: Отключение при множественных ошибках

### 📊 Мониторинг
- **Структурированное логирование**: Уровни DEBUG, INFO, ERROR
- **Метрики производительности**: Время операций, использование ресурсов
- **Детальные ошибки**: Контекст и stack trace
- **Аудит действий**: Все операции логируются

## 🚨 Решение проблем

### Ключи не выдаются после оплаты?
**✅ ИСПРАВЛЕНО**: Теперь ключи создаются автоматически сразу после успешной оплаты YooKassa.

### Бот падает при ошибках?
**✅ ИСПРАВЛЕНО**: Добавлена graceful degradation - бот продолжает работать при сбоях внешних сервисов.

### Медленная работа?
**✅ ОПТИМИЗИРОВАНО**: Все операции асинхронные, добавлены кэширование и connection pooling.

- `bot.log` - основной лог работы
- `.env` - конфигурация
- `vpnbot.db` - база данных

### Частые проблемы и решения
| Проблема | Решение |
|------------|----------|
| Бот не запускается | Проверьте `.env` и токен |
| Ключи не создаются | Проверьте 3x-UI подключение |
| Webhook не работает | Проверьте порт 8080 и URL |
| База данных ошибка | Проверьте права на запись |
